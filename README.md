# ♟ Polaris

This chess engine implements:
- Alpha-Beta Pruning for better performance
- Move Ordering for more branches to be pruned
- Quiescence Search to improve search results
- A command-line interface

## Installation

```
pip install -r requirements.txt
```

## Usage

### Interface

Start the interface with:

```
python3 interface.py
```

### UCI

Start the UCI (Universal Chess Interface) engine with:

```
python3 main.py
```
